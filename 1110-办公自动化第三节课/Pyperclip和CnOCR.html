Pyperclip模块安装

Pyperclip模块兼容python2和python3，能跨平台使用

pip install pyperclip

Pyperclip功能介绍 

pyperclip.copy() # 文本复制到剪贴板
pyperclip.paste() # 剪贴板粘贴文本,返回值为字符串类型

Pyperclip复制与粘贴 

# 复制文本到剪贴板
text = 'hello world'
pyperclip.copy(text)

# 从剪贴板粘贴文本
text = pyperclip.paste()
print(text)

除了常规的复制和粘贴操作外，Pyperclip还支持往剪贴板中添加内容，这个功能非常实用：

# 往剪贴板添加文本--ctrl+v会逐次递增
text = '你好世界'
pyperclip.copy(text + pyperclip.paste())

pyperclip剪贴板与文件的交互 

# 将文件读入内存
with open('123.txt', 'r') as f:
    content = f.read()
# 复制到剪贴板
pyperclip.copy(content)

或者，我们可以将剪贴板的内容写入文件：

# 从剪贴板获取内容
content = pyperclip.paste()
# 将内容写入文件
with open('example.txt', 'w') as f:
    f.write(content)

pyperclip的其他实用功能 

import random
import string

# 生成随机密码
def random_password(length=8):
    chars = string.ascii_letters + string.digits + string.punctuation
    return ''.join(random.choice(chars) for _ in range(length))

# 将密码复制到剪贴板
password = random_password()
pyperclip.copy(password)

什么是OCR

光学字符识别（Optical Character Recognition, OCR）是指对文本材料的图像文件进行分析识别处理，以获取文字和版本信息的过程。也就是说将图象中的文字进行识别，并返回文本形式的内容。

CnOCR

官方文档
CnOCR 是 Python 3 下的文字识别（Optical Character Recognition，简称OCR）工具包，支持简体中文、繁体中文（部分模型）、英文和数字的常见字符识别，支持竖排文字的识别。自带了20+个训练好的识别模型，适用于不同应用场景，安装后即可直接使用。同时，CnOCR也提供简单的训练命令供使用者训练自己的模型。

常见开源中文OCR

  • Paddle OCR   推荐指数（★★★★★）
  • CnOCR   推荐指数（★★★★★）
  • chinese_lite OCR   推荐指数（★★★）
  • EastOCR   推荐指数（★★）
  • Tesseract OCR   推荐指数（★）
  • chineseocr   推荐指数（★★）

CnOCR的安装 

查看[cnocr库安装使用.zip]，并根据安装说明书进行操作

CnOCR优点

  1. 轻量模型，执行速度快，速度略快于paddle ocr
  2. 支持pip直接安装
  3. ocr识别效果好，识别效果比肩paddle ocr
  4. 支持训练自己的模型

CnOCR缺点

1.部分符号识别效果差
2.部分场景下会出现空格丢失情况
3.模型补充训练没有paddle ocr方便

模型参数说明

模型参数

常见的图片识别 

from cnocr import CnOcr

img_fp = './docs/examples/huochepiao.jpeg'
ocr = CnOcr()  # 所有参数都使用默认值
out = ocr.ocr(img_fp)
print(out)

排版简单的印刷体截图图片识别 

from cnocr import CnOcr

img_fp = './docs/examples/multi-line_cn1.png'
ocr = CnOcr(det_model_name='naive_det')  # det_model_name检测模型名称
out = ocr.ocr(img_fp)
print(out)

竖排文字识别 

from cnocr import CnOcr

img_fp = './docs/examples/shupai.png'
ocr = CnOcr(rec_model_name='ch_PP-OCRv3') #rec_model_name识别模型名称
out = ocr.ocr(img_fp)
print(out)

英文识别 

from cnocr import CnOcr

img_fp = './docs/examples/en_book1.jpeg'
ocr = CnOcr(det_model_name='en_PP-OCRv3_det', rec_model_name='en_PP-OCRv3')
out = ocr.ocr(img_fp)
print(out)

繁体中文识别 

from cnocr import CnOcr

img_fp = './docs/examples/fanti.jpg'
ocr = CnOcr(rec_model_name='chinese_cht_PP-OCRv3')  # 识别模型使用繁体识别模型
out = ocr.ocr(img_fp)
print(out)

使用此模型时请注意以下问题：

  • 识别精度一般，不是很好；
  • 除了繁体字，对标点、英文、数字的识别都不好；
  • 此模型不支持竖排文字的识别。

单行文字的图片识别

如果明确知道待识别的图片是单行文字图片（如下图），可以使用类函数 CnOcr.ocr_for_single_line() 进行识别。这样就省掉了文字检测的时间，速度会快一倍以上

调用示例代码：
from cnocr import CnOcr
img_fp = './docs/examples/helloworld.jpg'
ocr = CnOcr()
out = ocr.ocr_for_single_line(img_fp)
print(out)
